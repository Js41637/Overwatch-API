Overwatch API

Returns an Overwatch Players Stats

API Endpoints

User Info:
/api/u/<user>

Returns basic user info

Params:

  • <user>
    - users battletag, case sensitive. e.g. Apples-1234
  • ?region=
    - optional region of user (us, eu, kr) - if not specified it will attempt to find it

Method Response: Show

{ ok: true, data: { player: { battletag: 'Apples-1234', avatar: 'https://blzgdapipro-a.akamaihd.net/game/unlocks/0x0250000000000BAD.png', region: 'us', level: 15, rank: 1, comprank: 2512 } } }

User Stats:
/api/u/<user>/stats/<type>

Returns a users QuickPlay and/or Competitive stats (if any)

Params:

  • <user>
    - users battletag, case sensitive. e.g. Apples-1234
  • <type>
    - optional type, will return only data relating to type, not sending a type will default to
    both
    , valid types are
    quickplay
    ,
    competitive
    or
    both
  • ?region=
    - optional region of user (us, eu, kr) - if not specified it will attempt to find it

User Heroes:

Returns all heroes and their playtime sorted by playtime for both QuickPlay and/or Competitive (if any)

Params:

  • <user>
    - users battletag, case sensitive. e.g. Apples-1234
  • <type>
    - optional type, will return only data relating to type, not sending a type will default to
    both
    , valid types are
    quickplay
    ,
    competitive
    or
    both
  • ?region=
